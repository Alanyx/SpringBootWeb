## 部署

1、打包

```
mvn clean package
```

2、数据库

```
数据库名:springboot
数据表：sysuser
```


3、环境变量

```
export MYSQL_DB_HOST=""
export MYSQL_DB_PORT=""
export MYSQL_DB_NAME=""
export MYSQL_DB_USERNAME=""
export MYSQL_DB_PASSWORD=""
```

4、运行

启动端口在：8066
